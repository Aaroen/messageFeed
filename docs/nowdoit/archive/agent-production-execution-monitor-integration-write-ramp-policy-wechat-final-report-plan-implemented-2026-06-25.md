# Agent 生产执行监控联调写放量策略与企微最终汇报计划

**创建日期**：2026-06-25

## 1. 本轮目标

在 Agent 已具备 Web/企业微信任务入口、实时进度、企业微信按钮直接控制、生产发布执行准备、外部监控配置映射、写能力小流量放量、企业微信验收签收和运营交接摘要的基础上，推进生产真实执行记录、外部监控平台联调状态、写能力放量策略参数化和企业微信最终汇报闭环，使 Agent 进一步接近可上线运行状态。

## 2. 实施范围

1. 生产真实执行记录：
   - 将生产发布执行准备推进为真实执行记录摘要。
   - 覆盖执行批次、执行人、执行状态、回滚门禁、失败退出、通知和审计证据。
2. 外部监控平台联调：
   - 将监控配置映射推进为外部平台联调状态摘要。
   - 覆盖指标写入、事件写入、告警通道、日报通道和联调结果。
3. 写能力放量策略参数化：
   - 对 `agent.schedule_message` 与 `agent.schedule_task` 建立可查询的放量策略参数摘要。
   - 覆盖放量比例、用户范围、审批门禁、预算门禁、回滚阈值和默认拒绝策略。
4. 企业微信最终汇报闭环：
   - 将企业微信验收签收推进为完成后最终汇报摘要。
   - 覆盖任务完成通知、最终报告入口、失败说明、用户下一步和审计证据。
5. 上线运行总览：
   - 汇总生产执行、监控联调、写放量策略和企业微信最终汇报状态。

## 3. 非目标

- 本轮不开放未进入白名单的写 capability。
- 本轮不移除企业微信文本 fallback。
- 本轮不执行不可回滚的真实生产变更。
- 本轮不绑定单一外部监控 SaaS 的私有 API。

## 4. 验收标准

- 生产真实执行记录可在 Web 或 service 层查看，并写入审计事件。
- 外部监控平台联调状态可查询指标、事件、通道和联调结果。
- 写能力放量策略参数摘要保持最小权限、审批、预算、审计和回滚约束。
- 企业微信最终汇报摘要覆盖完成通知、最终报告和失败说明。
- 上线运行总览纳入生产执行、监控联调、写放量策略和企微最终汇报状态。
- `go test ./...`、`go vet ./...`、`npm --prefix web run test`、`npm --prefix web run type-check`、`npm --prefix web run build` 通过。

## 6. 实施结果

1. 后端任务工作台已新增五类上线运行摘要：
   - `production_execution`：生产真实执行记录摘要，覆盖执行批次、执行人、执行状态、回滚门禁、失败退出、通知和审计证据。
   - `monitor_integration`：外部监控平台联调摘要，覆盖指标写入、事件写入、告警通道、日报通道和联调结果。
   - `write_ramp_policy`：写能力放量策略参数摘要，覆盖放量比例、用户范围、审批门禁、预算门禁、审计门禁、回滚阈值和默认拒绝策略。
   - `wechat_final_report`：企业微信最终汇报摘要，覆盖任务完成通知、最终报告入口、失败说明、用户下一步和审计证据。
   - `launch_runtime_overview`：上线运行总览摘要，覆盖生产执行、监控联调、写放量策略、企微最终汇报和审计状态。
2. 后端审计已新增五类快照事件：
   - `agent.production_execution_snapshot`
   - `agent.monitor_integration_snapshot`
   - `agent.write_ramp_policy_snapshot`
   - `agent.wechat_final_report_snapshot`
   - `agent.launch_runtime_overview_snapshot`
3. 前端任务工作台已新增上线运行展示：
   - API 类型定义已同步新增五类响应结构。
   - `AgentPlanView.vue` 已展示生产执行、监控联调、写放量策略、企微最终汇报和上线运行总览摘要及检查项。
4. 服务层测试已覆盖新增响应字段和审计事件。
