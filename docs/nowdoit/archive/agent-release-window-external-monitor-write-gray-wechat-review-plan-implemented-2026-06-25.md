# Agent 发布窗口外部监控写灰度与企微验收复盘计划

**创建日期**：2026-06-25

## 1. 本轮目标

在 Agent 已具备 Web/企业微信任务入口、实时进度、企业微信按钮直接控制、企微端到端验收摘要、发布窗口准备摘要、写能力灰度扩容准备摘要和外部监控对接准备摘要的基础上，推进真实生产发布窗口执行模拟、外部监控接入落地、写能力灰度扩容复核和企业微信用户验收复盘，使 Agent 进一步接近可受控上线和持续运营状态。

## 2. 实施范围

1. 发布窗口执行模拟：
   - 将发布窗口准备摘要推进为可执行窗口记录。
   - 覆盖窗口开始、审批确认、执行状态、失败退出、回滚确认、通知和审计。
2. 外部监控接入落地：
   - 将外部监控对接准备推进为 service 层可查询的指标和事件映射摘要。
   - 覆盖健康、SLA、错误、成本、队列、worker、通知失败、按钮控制和日报发送。
3. 写能力灰度扩容复核：
   - 对 `agent.schedule_message` 与 `agent.schedule_task` 的灰度执行结果进行扩容前复核。
   - 明确继续拒绝其他写 capability 的证据。
4. 企业微信用户验收复盘：
   - 输出企微发起任务、按钮控制、Web 同步查看、最终报告和失败回退路径的复盘摘要。
   - 覆盖用户可理解的状态、下一步和审计证据。
5. 运营日报闭环：
   - 将日报发送、监控告警、按钮控制和发布窗口状态纳入运营日报摘要。

## 2.1 本轮实施清单

   - 新增运营日报闭环摘要，覆盖日报发送、监控告警、按钮控制、发布窗口状态和审计证据。

## 3. 非目标

- 本轮不开放任意未列入白名单的写 capability。
- 本轮不移除企业微信文本 fallback。
- 本轮不强制连接真实外部监控 SaaS。
- 本轮不执行不可回滚的真实生产发布。

## 4. 验收标准

- 发布窗口执行模拟可在 Web 或 service 层查看，并写入审计事件。
- 外部监控接入摘要可查询指标、事件和通知通道映射。
- 写能力灰度扩容复核保持最小权限和默认拒绝策略。
- 企业微信用户验收复盘覆盖发起、进度、按钮控制、Web 同步、最终报告和失败回退。
- 运营日报摘要纳入发布窗口、监控告警、按钮控制和日报发送状态。
- `go test ./...`、`go vet ./...`、`npm --prefix web run test`、`npm --prefix web run type-check`、`npm --prefix web run build` 通过。

## 5. 后续衔接

本轮完成后，继续推进真实生产发布执行、外部监控平台真实配置、写能力小流量放量和企业微信用户验收签收。

## 6. 实施结果

1. 后端任务工作台已新增五类运行态摘要：
   - `release_window_execution`：发布窗口执行模拟摘要，覆盖窗口状态、审批确认、执行状态、失败退出、回滚确认、通知和审计事件。
   - `external_monitor_runtime`：外部监控运行态摘要，覆盖健康、SLA、错误、成本、队列、worker、通知失败、按钮控制和日报发送映射。
   - `write_gray_review`：写能力灰度扩容复核摘要，覆盖 `agent.schedule_message`、`agent.schedule_task`、默认拒绝、审批、预算、审计、回滚和拒绝模式。
   - `wechat_acceptance_review`：企业微信用户验收复盘摘要，覆盖企微入口、实时进度、按钮控制、Web 同步、最终报告和失败回退。
   - `operations_daily_closure`：运营日报闭环摘要，覆盖日报发送、监控告警、按钮控制、发布窗口状态和审计证据。
2. 后端审计已新增五类快照事件：
   - `agent.release_window_execution_snapshot`
   - `agent.external_monitor_runtime_snapshot`
   - `agent.write_gray_review_snapshot`
   - `agent.wechat_acceptance_review_snapshot`
   - `agent.operations_daily_closure_snapshot`
3. 前端任务工作台已新增运行态展示：
   - API 类型定义已同步新增五类响应结构。
   - `AgentPlanView.vue` 已展示发布窗口执行、外部监控运行、写灰度复核、企微验收复盘和运营日报闭环摘要及检查项。
4. 服务层测试已覆盖新增响应字段和审计事件。
