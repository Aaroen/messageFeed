# Agent 生产发布外部监控配置写放量与企微签收计划

**创建日期**：2026-06-25

## 1. 本轮目标

在 Agent 已具备 Web/企业微信任务入口、实时进度、企业微信按钮直接控制、发布窗口执行模拟、外部监控运行态摘要、写能力灰度复核、企业微信验收复盘和运营日报闭环摘要的基础上，推进生产发布执行准备、外部监控真实配置映射、写能力小流量放量和企业微信验收签收，使 Agent 上线前状态具备更明确的执行证据和用户确认路径。

## 2. 实施范围

1. 生产发布执行准备：
   - 将发布窗口执行模拟推进为生产发布执行记录。
   - 覆盖执行批次、批准来源、执行前检查、执行中状态、回滚门禁、通知和审计。
2. 外部监控真实配置映射：
   - 将外部监控运行态摘要推进为可配置的监控目标映射。
   - 覆盖指标名、事件名、告警通道、日报通道和外部平台配置状态。
3. 写能力小流量放量：
   - 对 `agent.schedule_message` 与 `agent.schedule_task` 建立小流量放量摘要。
   - 保持其他写 capability 默认拒绝或需审批。
4. 企业微信验收签收：
   - 将企业微信验收复盘推进为可审计的签收摘要。
   - 覆盖发起任务、查看进度、按钮控制、Web 同步、最终报告和失败回退确认。
5. 运营可交接性：
   - 将生产发布、外部监控、写放量和企微签收纳入运营交接摘要。

## 3. 非目标

- 本轮不开放未进入白名单的写 capability。
- 本轮不移除企业微信文本 fallback。
- 本轮不执行不可回滚的真实生产变更。
- 本轮不强制依赖特定外部监控 SaaS。

## 4. 验收标准

- 生产发布执行准备可在 Web 或 service 层查看，并写入审计事件。
- 外部监控配置映射可查询指标、事件、通道和配置状态。
- 写能力小流量放量摘要继续保持最小权限、审批、预算和审计约束。
- 企业微信验收签收摘要覆盖用户确认路径和失败回退路径。
- 运营交接摘要纳入发布、监控、写放量和签收状态。
- `go test ./...`、`go vet ./...`、`npm --prefix web run test`、`npm --prefix web run type-check`、`npm --prefix web run build` 通过。

## 6. 实施结果

1. 后端任务工作台已新增五类上线前交接摘要：
   - `production_release`：生产发布执行准备摘要，覆盖执行批次、批准来源、执行前检查、执行中状态、回滚门禁、通知和审计。
   - `external_monitor_config`：外部监控配置映射摘要，覆盖指标名、事件名、告警通道、日报通道和外部平台配置状态。
   - `write_ramp`：写能力小流量放量摘要，覆盖 `agent.schedule_message`、`agent.schedule_task`、放量比例、默认拒绝、审批、预算、审计和回滚门禁。
   - `wechat_signoff`：企业微信验收签收摘要，覆盖发起任务、查看进度、按钮控制、Web 同步、最终报告和失败回退确认。
   - `operations_handoff`：运营交接摘要，覆盖生产发布、外部监控配置、写放量、企业微信签收和审计交接。
2. 后端审计已新增五类快照事件：
   - `agent.production_release_snapshot`
   - `agent.external_monitor_config_snapshot`
   - `agent.write_ramp_snapshot`
   - `agent.wechat_signoff_snapshot`
   - `agent.operations_handoff_snapshot`
3. 前端任务工作台已新增上线前交接展示：
   - API 类型定义已同步新增五类响应结构。
   - `AgentPlanView.vue` 已展示生产发布、监控配置、写能力放量、企业微信签收和运营交接摘要及检查项。
4. 服务层测试已覆盖新增响应字段和审计事件。
