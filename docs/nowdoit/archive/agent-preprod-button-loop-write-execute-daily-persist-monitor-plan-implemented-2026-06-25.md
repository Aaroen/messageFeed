# Agent 预发布按钮闭环写能力执行日报持久化与上线监控计划

**创建日期**：2026-06-25

## 1. 本轮目标

在 Agent 已具备 Web/企业微信任务入口、实时进度、任务工作台、企业微信原生发送 payload、按钮联调摘要、写能力灰度策略、写操作回放、上线审批和生产日报摘要的基础上，推进真实生产预发布、按钮交互闭环实测、写能力最小权限执行、生产日报持久化和上线后监控，使 Agent 进一步进入可预发布、可实测、可受控执行和可持续运营阶段。

## 2. 实施范围

1. 真实生产预发布：
   - 输出预发布环境验收摘要。
   - 覆盖配置、迁移、健康检查、worker、Web 入口、企业微信入口、通知、告警、回滚和审计。
2. 按钮交互闭环实测：
   - 输出企业微信按钮点击闭环摘要。
   - 覆盖查看进度、审批、重试、恢复、取消和最终报告动作。
   - 保留文本 fallback。
3. 写能力最小权限执行：
   - 针对 `agent.schedule_message` 与 `agent.schedule_task` 输出最小权限执行摘要。
   - 覆盖审批、预算、权限、执行、审计和回滚。
   - 默认拒绝其他写能力。
4. 生产日报持久化：
   - 将生产日报从即时摘要推进到可回溯持久记录。
   - 覆盖任务量、成功率、失败、告警、成本、趋势、接管、恢复和通知。
5. 上线后监控：
   - 输出上线后监控摘要。
   - 覆盖健康、SLA、错误、告警、成本、队列、worker、通知和人工接管。

## 3. 非目标

- 本轮不执行真实生产发布。
- 本轮不开放任意写操作 capability。
- 本轮不移除文本 fallback。
- 本轮不接入外部监控平台。

## 4. 验收标准

- Web 或 service 层可以查看预发布验收摘要。
- 企业微信按钮闭环摘要覆盖查看进度、审批、重试、恢复、取消和最终报告。
- 写能力最小权限执行摘要默认受审批、预算、权限、审计和回滚约束。
- 生产日报具备可回溯记录摘要。
- 上线后监控摘要覆盖健康、SLA、错误、告警、成本、队列、worker、通知和接管。
- `go test ./...`、`go vet ./...`、`npm --prefix web run test`、`npm --prefix web run type-check`、`npm --prefix web run build` 通过。

## 5. 本轮实施清单

1. 后端任务工作台响应补充预发布验收摘要、企业微信按钮闭环摘要、写能力最小权限执行摘要、生产日报持久化摘要和上线后监控摘要。
2. 预发布验收摘要覆盖配置、迁移、健康检查、worker、Web 入口、企业微信入口、通知、告警、回滚和审计。
3. 企业微信按钮闭环摘要覆盖查看进度、审批、重试、恢复、取消和最终报告动作，并保留文本 fallback。
4. 写能力最小权限执行摘要覆盖审批、预算、权限、执行、审计和回滚，默认拒绝其他写能力。
5. 生产日报持久化摘要基于生产日报快照输出可回溯记录状态。
6. 上线后监控摘要覆盖健康、SLA、错误、告警、成本、队列、worker、通知和人工接管。
7. Web API 类型和任务工作台展示新增摘要，补充测试并执行完整验证。

## 7. 实施结果

1. 后端任务工作台响应已新增 `preprod`、`button_loop`、`write_execute`、`daily_persist` 和 `post_launch_monitor` 五类摘要。
2. 预发布验收摘要已覆盖配置、迁移、健康检查、worker、Web 入口、企业微信入口、通知、告警、回滚、审计、真实联调和运维验收。
3. 企业微信按钮闭环摘要已覆盖查看进度、审批、重试、恢复、取消、最终报告和文本 fallback。
4. 写能力执行摘要已基于写操作回放与最小权限策略输出候选 capability、默认拒绝策略、审批、预算、权限、执行、审计和回滚触发条件。
5. 生产日报持久化摘要已基于生产日报快照生成可回溯记录键，并通过审计事件保留。
6. 上线后监控摘要已覆盖健康、SLA、错误、告警、成本、队列、worker、通知、人工接管和趋势。
7. Web API 类型和任务工作台已展示新增摘要，服务层测试已覆盖新增字段和审计事件。
